# Changelog

## 0.6.0 (2026-03-11)

### Теги и избранное

- Новая система тегов для организации библиотеки
- Создание тегов с иконками-эмодзи или цветными маркерами
- Диалог создания тега: переключатель Adw.ToggleGroup между режимами эмодзи и цвета, встроенный выбор иконки в строке ввода, палитра цветов с кольцевым выделением и подсветкой при наведении, проверка дублирования имён, автогенерация имени для цветовых тегов
- Назначение тегов релизам через кнопку-звезду на странице релиза (SplitButton с выпадающим поповером)
- Поиск по тегам в поповере назначения, сообщение «Тег не найден» с подсказкой создания
- Вкладка «Избранное и теги» с режимами карточек и списка, переключение сохраняется между сессиями
- Бейджи тегов на карточках каталога (до 3 иконок + «+» для остальных)
- Интерактивные пилюли тегов на странице релиза с навигацией к содержимому тега
- Удаление тегов из хедер-бара и списочного режима с диалогом подтверждения
- Мгновенное обновление бейджей на всех видимых карточках при изменении тегов
- Статистика тегов и кнопка очистки в настройках

### Прогресс просмотра серий

- Полосы прогресса на карточках и строках серий, показывающие долю просмотра
- Отметка просмотренных серий галочкой акцентного цвета
- Размытие превью непросмотренных серий (настраивается в параметрах)
- Фильтрация серий по статусу просмотра (просмотренные/непросмотренные)
- Спиннер загрузки и иконка-заглушка для превью серий
- Автоматическое обновление прогресса при возврате из плеера

### Панель фильтров

- Диалог фильтров заменён на выдвижную боковую панель (OverlaySplitView)
- Автоприменение фильтров с задержкой 1 секунда
- Асинхронная загрузка жанров и диапазона годов
- Подавление лишних перезагрузок каталога при неизменённых фильтрах

### Навигация по жанрам

- Клик на жанр на странице релиза открывает релизы этого жанра как NavigationPage
- Кнопка «назад» возвращает на страницу релиза
- Эффект наведения на чипах жанров

### Страница релиза

- Пустые вкладки (связанные релизы, команда, торренты) скрываются автоматически
- Вкладки появляются динамически при получении асинхронных данных

### Визуальные улучшения

- Плавные CSS-анимации наведения (200ms ease-out-quad) на элементах каталога, боковой панели, вкладках навигации, кнопках плеера и чипах фильтров
- Полоса прогресса просмотра увеличена до 4px с акцентным цветом
- Карточки тегов с центрированными заголовками и размытым эмодзи-фоном
- Цветные круги тегов с обводкой для лучшей видимости

### Плеер

- Исправлена очистка плеера при навигации назад — убраны ненадёжные сигналы hiding/unmap
- Очистка происходит только через кнопку «назад», сигнал NavigationView.popped и закрытие окна
- Идемпотентная очистка предотвращает двойную остановку

### Инфраструктура

- Централизация URL-адресов API AniLibria в единый файл констант
- Лимит загрузки изображений увеличен с 10 МБ до 30 МБ
- Репозиторий перенесён в организацию alt-gnome
- Добавлены Telegram-канал и чат проекта
